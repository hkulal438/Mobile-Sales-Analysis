**Motorola Mobile Sales Dashboard**
A dynamic dashboard for visualizing and analyzing Motorola mobile sales data. It provides interactive charts, city-level mapping, and insights by brand, model, payment method, and time period.

Features
Total sales, transaction count, and quantity metrics

Filter by mobile model, payment method, brand, and day

City-wise sales mapping across India

Monthly sales quantity trends

Customer ratings distribution

Payment method breakdown

Brand and model-wise sales performance

Day-wise sales analysis
